info threads 查看所有线程信息
thread find 查找线程
thread num 切换线程
b breakpoint thread id 为线程设置端点
thread apply 为线程执行命令
set scheduler-locking off/on/step 锁定线程

thread 0x7ff..(线程地址) LWP(线程pid，light weight process) at (在某一行) in （在哪个函数停下来)
